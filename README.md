# 런어스 백업 툴 (LearnUs Backup Tool)

연세대학교 학습 관리 시스템 **LearnUs(런어스)**의 강의 자료(동영상, 파일, 과제, 공지사항)를 자동으로 아카이빙하고, 오프라인에서 편리하게 열람할 수 있는 백업 도구입니다.

## 주요 기능 (Features)

*   **강의 자동 탐색**: 현재 학기의 수강 강좌를 자동으로 감지합니다.
*   **동영상 백업**: `ffmpeg`를 사용하여 강의 동영상을 고화질로 다운로드합니다.
*   **자료 통합 백업**: 강의 자료, 과제(제출 파일 포함), 공지사항(첨부파일 포함)을 원본 구조 그대로 저장합니다.
*   **데이터 기반 저장**: 모든 메타데이터는 JSON 형식으로 저장되어 데이터와 뷰어가 분리되어 있습니다.
*   **오프라인 뷰어**: 인터넷 연결 없이도 다운로드한 자료를 깔끔한 웹 인터페이스(Bootstrap 디자인)로 열람할 수 있습니다.

## 사전 요구 사항 (Prerequisites)

이 프로그램을 실행하기 위해서는 다음이 필요합니다:
1.  **Python 3.8 이상**: [Python 다운로드](https://www.python.org/downloads/)
2.  **FFmpeg**: 동영상 다운로드를 위해 필수입니다.
    *   **Mac**: 터미널에서 `brew install ffmpeg` 입력 (Homebrew 필요)
    *   **Windows**: [ffmpeg.org](https://ffmpeg.org/)에서 다운로드 후 압축 해제, `bin` 폴더를 환경 변수 Path에 추가.

## 설치 및 실행 방법 (Installation & Usage)

복잡한 설정 없이 아래 스크립트를 실행하면 가상 환경(venv)을 자동으로 생성하고 필요한 라이브러리를 설치합니다.
(처음 한 번만 실행하면 됩니다.)

### Mac / Linux 사용자
1. 터미널을 열고 폴더로 이동합니다.
2. 다음 명령어를 실행하세요:
   ```bash
   chmod +x run_mac.sh
   ./run_mac.sh
   ```

### Windows 사용자
1. 폴더 내의 `run_win.bat` 파일을 더블 클릭하세요.
2. 자동으로 설치 및 실행이 진행됩니다.

## 사용 가이드 (How to Use)

1.  **로그인**: 프로그램이 시작되면 브라우저 창이 열리고 런어스 로그인 페이지가 뜹니다. 직접 로그인을 완료해주세요.
2.  **강좌 선택**: 로그인이 완료되면 백업할 강좌 목록이 표시됩니다. 원하는 강좌를 선택하세요.
3.  **아카이빙**: 프로그램이 자동으로 자료를 수집합니다. 진행 상황은 터미널에 표시됩니다.
4.  **뷰어 실행**:
    *   다운로드가 완료된 후, 백업된 자료를 보려면 뷰어를 실행해야 합니다.
    *   (현재 제공된 스크립트는 수집기(`main.py`)를 실행합니다. 뷰어는 아래 방법으로 실행하세요.)
    
    **뷰어 실행 방법**:
    *   **Mac**: 터미널에서 `source venv/bin/activate` 입력 후 `python viewer.py` 실행
    *   **Windows**: 터미널에서 `venv\Scripts\activate` 입력 후 `python viewer.py` 실행
    *   브라우저에서 `http://localhost:5000` 주소로 접속하면 뷰어가 열립니다.


## 보안 및 책임 (Security & Disclaimer)

*   **로컬 구동**: 이 프로그램은 사용자의 컴퓨터(Local Machine)에서만 작동합니다. 어떠한 데이터도 외부 서버로 전송되지 않습니다.
*   **오픈 소스**: 모든 소스 코드는 투명하게 공개되어 있습니다. 보안상 위험한 코드가 없음을 직접 확인하실 수 있습니다.
*   **사용자 책임**: 본 도구는 개인적인 편의를 위해 제작되었으며, 개인 컴퓨터에서 실행하는 만큼 프로그램 사용으로 인해 발생하는 모든 책임은 사용자 본인에게 있습니다.
*   **저작권 준수**: 다운로드한 강의 자료는 **개인 소장용**으로만 사용해야 하며, 타인과 공유하거나 복제/배포할 경우 저작권법 및 학교 규칙에 의해 처벌받을 수 있습니다.
